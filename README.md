The begginnning of my climbing tracker app.

- Track climbing progress
- View climbing history

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/climbing-tracker-app.git
2. Install dependencies
   ```bash
   npm install
3. Run the app
   ```bash
   npx expo start
